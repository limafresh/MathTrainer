# Math trainer  

An online math trainer for kids. It includes:

- Addition within 100
- Multiplication within 10
- Addition within 1000

Launch: ➡️ https://limafresh.github.io/MathTrainer/

![screenshot](https://raw.githubusercontent.com/limafresh/MathTrainer/main/screenshot.png)

## How to use  
The application generates an example with random numbers. You need to enter the correct answer to it and click the "CHECK" button.

## Languages
- English
- Ukrainian
- Russian

In the future we will add other languages. You can ask to add the desired language in Disscussions, I will add it faster.
